# Examples

This directory contains example scripts demonstrating various features of bob.bio.vein.

## Dorsal Hand Vein Recognition

**dorsalhandveins_example.py** - Demonstrates how to work with dorsal hand vein databases and ROI annotations.

Run this example:
```bash
python examples/dorsalhandveins_example.py
```

This example shows:
- Basic database usage
- Working with ROI annotations
- Applying preprocessing pipelines

For more detailed information, see `doc/DORSAL_HAND_VEIN_ROI.md`.
